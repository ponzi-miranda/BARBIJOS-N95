Pasos para lanzar FRONT

1. Ir a la carpeta client/

Pasos para lanzar BACKEND

1. Ir a la carpeta server/
2. En una terminal tirar npm run build (build + auto refresh)
3. En otra terminal tirar npm run dev (servidor)
